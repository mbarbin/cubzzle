Module Cubzzle.Rotation

Encoding ways of rotating a piece in space.

type t
val sexp_of_t : t -> Sexplib0.Sexp.t
val cardinality : Base.int

The number of different rotations.

Rotations are ordered and indexed. The index may serve as efficient encoding.

val to_index : t -> Base.int

to_index t returns the index of t in the total code set. It is guaranteed that 0 <= index < cardinality.

val of_index_exn : Base.int -> t

of_index index returns the code at the given index. Indices are expected to verify 0 <= index < cardinality. An invalid index will cause the function to raise.

val apply : t -> Coordinate.t -> Coordinate.t

Apply the given rotation to a set of coordinates.
