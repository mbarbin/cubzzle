Module Cubzzle.Box

type t

The enclosing box, defined by its size. It must be exactly big enough to hold all the components of the pieces when they're in place.

The box is mutable - one can push and pop pieces from the box.

val create : goal:Z_shape.t -> t
val size : t -> Cubzzle__Size.t
val is_available : t -> Cubzzle__Coordinate.t -> bool

Returns true if the given cooridinate is inside the box and currently not occupied by any piece.

val contents : t -> Cubzzle__Coordinate.t -> Cubzzle__Piece.t option
module Stack : sig ... end

The box acts as a stack of pieces. One can push and pop pieces from the box.

val print_floors : t -> unit

Print the contents of the box on stdout, floor by floor. This starts by the top most floor and goes on for decreasing values of z.
